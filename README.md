### 介绍
基于DDD分层实现的web版 **linux(终端 文件 脚本 进程)、数据库（mysql postgres）、redis(单机 集群)、mongo统一管理操作平台**


### 开发语言与主要框架
- 前端：typescript、vue3、element-plus
- 后端：golang、gin、gorm
